# Get Started

## Installation

<!-- tabs:start -->

#### **Npm**

```bash

npm i @gray-adeyi/paystack-sdk
```

#### **Yarn**

```bash

yarn i @gray-adeyi/paystack-sdk
```

#### **Pnpm**

```bash

pnpm i @gray-adeyi/paystack-sdk
```

#### **Bun**

```bash

bun add @gray-adeyi/paystack-sdk
```

#### **Deno**

```bash

deno add @gray-adeyi/paystack-sdk
```
